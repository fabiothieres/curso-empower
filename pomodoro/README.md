# 🍅 Pomodoro Web  

Projeto desenvolvido para praticar **HTML5** e **SCSS**, inspirado na técnica **Pomodoro**, criada por Francesco Cirillo.  
O objetivo é apresentar de forma visual e simples como funciona esse método de produtividade muito usado por programadores e estudantes.

---

## 🚀 Tecnologias utilizadas
- HTML5  
- SCSS  
- Google Fonts  

---

## 🎯 Objetivo do projeto
O **Pomodoro Web** tem como foco a prática de conceitos fundamentais de front-end, como:  
- Estruturação semântica do HTML  
- Organização e aninhamento no SCSS  
- Criação de um layout visual limpo e responsivo  
- Uso de imagens e tipografia para reforçar o design  

---

## 🧠 O que é a técnica Pomodoro?
A técnica Pomodoro é um método de gestão de tempo que divide o trabalho em blocos de 25 minutos de foco (chamados *pomodoros*), seguidos de pequenas pausas.  
Após quatro pomodoros, é feita uma pausa maior para descanso.
